About

I've always loved building things, from Lego's to apps to rockets. I think creating something from nothing is one of my favorite experiences. Luckily, I was able to find my passion in building software early on. Every day I'm still amazed that I'm able to tell pieces of sand how to do math.
I started my software journey by learning the Swift language, and building some simple apps for myself. I then learned Java and Python through high school classes. In university, I've used C/C++ the most and become really interesting in low-level programming.
Personally, I'm very organized, sociable, and a fast-learner. I spend most of my time learning new things, whether that's in class or for fun. I try to learn a new language every year and build some sort of personal project with it. I learned Rust a few years ago and built a compiler for an open source project. I'm in the process of learning Assembly because it's vastly different than anything I know already.
When I'm not programming, in class, or catching up on sleep, I'm usually outside running or skateboarding. I also spend time listening to 80s synth rock and reading old science fiction novels.
